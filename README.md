## 中文
查找自第一天早上恢复的数据库。

## English
Find databases that were restored since the first morning.

---
## Logic
```SQL
use msdb;
set nocount on

select
	'restored by' 	= user_name
, 	destination_database_name
, 	restore_date
from
	msdb..restorehistory
where
	restore_date > '2014-05-12 00:00:00:000'
order by
	restore_date desc


```
